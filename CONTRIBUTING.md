# Contributing

Thanks for your interest in improving **SwasthyaSetu Wellness Guide**.

## Ways to contribute

- Improve Hindi content clarity and grammar.
- Fix layout or responsive design issues.
- Add accessibility improvements (ARIA labels, focus styles, semantic structure).
- Enhance SEO metadata and performance.

## Suggested workflow

1. Create a new branch from `work`.
2. Make focused changes with clear commit messages.
3. Run a quick local check in browser.
4. Open a PR with:
   - What changed
   - Why it changed
   - Screenshots (if visual change)

## Content safety note

This project discusses health topics. Avoid adding absolute medical claims and prefer evidence-based, cautionary wording.
