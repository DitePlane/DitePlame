# Contributing

Thanks for contributing to **SwasthyaSetu Wellness Guide**.

## Scope

This repository is a static educational web page. Please keep contributions aligned with:

- Health education clarity
- Respectful and non-judgmental tone
- Simple, maintainable front-end code

## How to contribute

1. Create a branch for your change.
2. Make focused edits.
3. Preview the page locally.
4. Open a PR with a short summary and screenshots (if UI changed).

## Content guidelines

- Avoid fear-based or stigmatizing language.
- Do not add explicit/graphic content.
- Keep claims moderate; avoid absolute guarantees.
- Include disclaimers when adding sensitive health guidance.

## Technical guidelines

- Keep everything runnable as a static page.
- Test in a modern browser after changes.
- Maintain responsive behavior for mobile and desktop.
- Prefer small, readable JavaScript helpers.

## Commit style

Use concise commit messages that explain intent, for example:

- `docs: add getting started instructions`
- `content: refine FAQ wording`
- `style: improve mobile spacing in hero section`
