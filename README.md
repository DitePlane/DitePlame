# SwasthyaSetu Wellness Guide

A single-page Hindi wellness website focused on sexual health recovery, nutrition, exercise, myths vs facts, mental wellness, and guidance on when to consult a doctor.

## What this project includes

- A responsive static website (`index.html`) with complete layout, styles, and content.
- SEO-focused metadata and social preview tags.
- Hindi-first content and typography for readability.

## Quick start

Because this is a static site, you can run it locally in multiple ways.

### Option 1: Open directly

1. Clone/download the repo.
2. Open `index.html` in your browser.

### Option 2: Run a local HTTP server (recommended)

```bash
python3 -m http.server 8080
```

Then open: <http://localhost:8080>
