# SwasthyaSetu Wellness Guide

A static Hindi-first health education website focused on sexual wellness, recovery, nutrition, fitness, skin-care habits, myth-busting, mental wellbeing, and doctor-escalation guidance.

## Public Search Readiness (SEO)

This project includes:
- On-page SEO metadata (`description`, `keywords`, `robots`, Open Graph, Twitter, canonical).
- Structured data (`WebSite` and `FAQPage` JSON-LD).
- Crawl/index files: `robots.txt` and `sitemap.xml`.
- Long-form informational content for search discoverability.

## Files

- `index.html` — full website (single-page app-like static layout).
- `robots.txt` — crawler rules + sitemap location.
- `sitemap.xml` — canonical URL list.
- `AGENTS.md` — contribution and editing rules for AI/code agents.

## Run locally

```bash
python3 -m http.server 4173
```

Open: `http://127.0.0.1:4173`

## Publish notes

1. Deploy this repo to any static host (Netlify, Vercel static, GitHub Pages, Cloudflare Pages).
2. Keep canonical and sitemap domain aligned with your live domain.
3. Submit sitemap URL in Google Search Console and Bing Webmaster Tools.
4. Re-submit sitemap after major content updates.

## Content intent

- Educational purpose only.
- Not a replacement for diagnosis/treatment by licensed professionals.
- Designed for responsible, consent-aware, non-judgmental health education.
