# SwasthyaSetu Wellness Guide

A static, single-page Hindi wellness website focused on sexual health education, recovery habits, and practical daily lifestyle guidance.

## What this project includes

- A fully self-contained `index.html` file (HTML + CSS + JavaScript).
- Section-based educational content for:
  - Recovery routines
  - Nutrition and hydration
  - Exercise and sleep habits
  - Mental wellbeing and digital balance
  - FAQ and legal/disclaimer notes

## Quick start

Because this is a static site, you can run it locally in two easy ways.

### Option 1: Open directly

Open `index.html` in your browser.

### Option 2: Serve with a local HTTP server (recommended)

```bash
python3 -m http.server 8080
```

Then visit:

```text
http://localhost:8080
```

## Editing guide

- Keep content clear, respectful, and educational.
- Preserve mobile responsiveness when updating styles.
- Prefer semantic HTML and accessible labels (`aria-*`, descriptive buttons).
- Re-check FAQ toggle behavior after JavaScript edits.

## Important note

This site content is educational and should not be treated as medical diagnosis or treatment advice.
